About this project

MalawiUniGuide is a student-built website that helps prospective students explore higher education in Malawi. It demonstrates responsive design, accessibility, and JavaScript-driven dynamic content.

Features

  • 15+ dynamic university entries loaded from JSON
  • Search, filter, and favorites (localStorage)
  • Accessible modal dialogs
  • Contact form with action page
